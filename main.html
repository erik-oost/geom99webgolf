Problem


Ontario, Canada

Problem Statement

Ideal: An interactive map to display the municipalities within Ontario and links to their official websites.
Reality: There is currently no web map that provides the links to the websites for the municipalities within all of Ontario.
Consequences: Currently, a user would need to use a map to find the municipality they are looking for and then use Google to find the official website manually.
Proposal: An interactive web map which displays all of the municipalities within Ontario, with the ability for users to click on each municipality and see a popup that contains its name and a link to the official website.
Technology: Geocortex 5, ArcGIS Online

Solution


To solve the problem at hand, an interactive map was developed that displays all lower and upper tier municipalities in Ontario for the public to explore. Users can explore their boundaries and will have access to all their official names and websites just by clicking on the area of interest.
