Problem


Ontario, Canada

Problem Statement

Ideal: An interactive map to display the municipalities within Ontario and links to their official websites.
Reality: There is currently no web map that provides the links to the websites for the municipalities within all of Ontario.
Consequences: Currently, a user would need to use a map to find the municipality they are looking for and then use Google to find the official website manually.
Proposal: An interactive web map which displays all of the municipalities within Ontario, with the ability for users to click on each municipality and see a popup that contains its name and a link to the official website.
Technology: Geocortex 5, ArcGIS Online

Solution


To solve the problem at hand, an interactive map was developed using Geocortex 5 that displays all lower and upper tier municipalities in Ontario for the public to explore. Users can explore their boundaries and will have access to all their official names and websites just by clicking on the area of interest.

Upper and Lower Tier Municipalities of Ontario

Click the link above to view the final product. The link will be inactive beyond April 30, 2021. The image below is what the users will first see when they access the web map.

Tabs

The column on the left side of the web page shows three tabs labelled Home, Layers, and Legend. The home tab describes what users will be able to do with the features to view the municipalities. The layer tab shows the two layers called Lower Boundary and Upper Boundary that can be turned on or off to display them on the map. The third tab is the legend which shows which polygons belong to each layer. The upper boundary municipalities are coloured yellow and the lower boundary municipalities are blue.

Search Bar

Users can use the search bar to enter keywords to locate areas of interest and are provided with a list of results relating to the search. When the user clicks on one of these results, they are provided with the option to view the location with the World Geocoder for ArcGIS or with the Lower or Upper Boundary layers if applicable. For the example of Gravenhurst, the selectable layers based on this search are the World Geocoder for ArcGIS and the Lower Boundary layer.

Toolbar

The toolbar provides the users with the opportunity to draw points, lines or polygons on the map and then clear those markups with the click of a button. There is also a measure tool that allows the user to measure lengths across the map or measure any given area. The fourth option on the toolbar is the identify tool that allows the user to pinpoint a location on the map and results for municipalities in that area appear in a list. The list shows both municipality tier layers and has the options for users to click on a feature of interest and have it attributes appear in the side column. The last option in the toolbar is the ability to print the map at a scale determined by the user. With the I want to menu, users are able to return to the initial map position and zoom level and also switch the colour theme to dark depending on the user's preference.

Widgets

In the bottom right corner of the web page there are two widgets that help users customize the map. The button on the left allows the user to choose a new basemap from a provided list. The button on the right allows users to create bookmarks of an area and give it a name. With these bookmarks, the users are able to quickly return to that view and continue exploring. In the top left corner of the map under the I want to menu, there is another widget that allows the user to zoom in and out on the map. This can also be accomplished by scrolling with the mouse.
